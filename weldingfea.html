Welding is considered one of the oldest form of additive manufacturing process. I had a chance to work on solving a problem of analyzing gas arc welding performed on aluminum material.

I accomplished analyzing gas arc welding process on aluminum material. Solving this problem computationally saved thousands of dollars from the project budget which otherwise would be required for personnel training, equipment and time. I used Goldak's heat source model equations to create a moving heat source. Utilized HPC using FORTRAN language to get accurate and faster results.

The model was verified with the experimental studies performed and published. Using the data from the model we were able to accurately set up the experiment methodology for the project.
